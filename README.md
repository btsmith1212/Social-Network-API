# Social-Network-API

## Description

This is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. It uses Express.js for routing, a MongoDB database, and the Mongoose ODM. It also uses dayJS for timestamps, Express.js, and Mongoose packages.

## Installation

Run npm install to install modules, run to local server using npm run start.

## Usage

Submit queries using the designated routes in an API testing environment such as Thunderclient or Insomnia.
